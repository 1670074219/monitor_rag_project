# 删除人数为0的视频记录

## 📋 功能说明

这些脚本用于从 `video_description.json` 文件中删除分析结果显示"人数：0"的视频记录，以清理无人视频数据。

## 📁 脚本文件

### 1. `remove_zero_person_videos.py` - 直接删除版本
- **特点**：自动删除，无需用户确认
- **适用场景**：确定要删除且信任脚本的情况

### 2. `remove_zero_person_videos_interactive.py` - 交互式版本 ⭐推荐
- **特点**：预览删除列表，用户确认后删除
- **适用场景**：首次使用或需要确认删除内容

## 🚀 使用方法

### 方法1：交互式删除（推荐）

```bash
cd video_process
python remove_zero_person_videos_interactive.py
```

**执行流程：**
1. 📖 读取数据并统计
2. 🔍 查找人数为0的记录
3. 📊 显示预览结果和删除列表
4. ⚠️ 用户确认是否删除
5. 💾 备份原文件并执行删除

### 方法2：直接删除

```bash
cd video_process
python remove_zero_person_videos.py
```

**执行流程：**
1. 📖 读取数据
2. 💾 自动备份原文件
3. 🗑️ 直接删除人数为0的记录
4. ✅ 保存结果

## 🛡️ 安全特性

### ✅ 自动备份
- 删除前自动备份原始文件
- 备份文件名格式：`video_description.json.backup_YYYYMMDD_HHMMSS`
- 可以随时恢复数据

### ✅ 详细统计
- 显示原始记录数
- 显示删除记录数和比例
- 显示保留记录数

### ✅ 错误处理
- 文件不存在检查
- JSON格式验证
- 异常捕获和详细错误信息

## 🔄 数据恢复

如果需要恢复删除的数据：

```bash
# 查看备份文件
ls video_description.json.backup_*

# 恢复数据（替换为实际的备份文件名）
cp video_description.json.backup_20250623_143022 video_description.json
```

## ⚠️ 注意事项

1. **运行前确认**：确保当前目录为 `video_process`
2. **备份重要**：虽然脚本会自动备份，但建议额外备份重要数据
3. **网络存储**：如果数据在网络存储上，确保有足够权限和空间
4. **大文件处理**：大文件可能需要较长处理时间，请耐心等待

## 🆘 常见问题

**Q：如何只查看有多少人数为0的记录，不删除？**
A：使用交互式版本，在确认阶段选择 'n' 取消即可。

**Q：删除后发现数据有问题怎么办？**
A：使用备份文件恢复，备份文件名包含时间戳。

**Q：可以自定义删除条件吗？**
A：可以修改脚本中的 `'人数：0' in analyse_result` 条件。
